# Registration Form

A responsive user registration form developed using HTML and CSS. 
This project is designed to collect user details through a clean and user-friendly interface.

## Technologies Used

- HTML5
- CSS3

## Features

- User name input
- Email and password fields
- Gender selection using radio buttons
- Hobbies selection using checkboxes
- Dropdown selection
- Submit button
- Responsive design

## Project Purpose

The purpose of this project is to understand HTML form elements and CSS styling concepts to create an attractive and structured registration form.

## How to Run

1. Download or clone the repository
2. Open `registration-form.html` in any web browser
3. Fill in the form details
